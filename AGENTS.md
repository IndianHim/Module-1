# AGENTS

This repository is a small beginner Python lesson workspace.

## What this workspace contains
- Simple standalone Python scripts and exercises.
- Top-level folders: `Lesson 1`, `Lesson 2`, `Lesson 3`, and `freeplay code`.
- No package, no tests, no build system.

## How to work with this repository
- Use Python 3 to run files directly: `python3 "Lesson 1/First Python Program.py"`.
- Keep edits simple and beginner-friendly unless the user explicitly asks for more advanced changes.
- Preserve existing file names, folder names, and the overall structure.
- Do not add project scaffolding, dependencies, or packaging unless requested by the user.

## Notes for AI agents
- This repository is educational; the main goal is to help the user with Python learning exercises.
- Focus on clarity, correct syntax, and straightforward solutions.
- If the user asks for enhancements, explain any new concepts before introducing them.
